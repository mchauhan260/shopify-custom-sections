# Shopify Custom Sections
 
This repository contains production-ready Shopify custom sections
built for real client stores.
 
## Included Sections
- Custom Announcement Bar
- Custom Product Slider (coming soon)
 
## Features
- Shopify Online Store 2.0 compatible
- Clean Liquid code
- Schema-based admin settings
- Mobile responsive
- Performance friendly
 
## Usage
Copy the section file into:
Online Store → Themes → Edit code → Sections
